# Vtrack Node.Js Library

[![Version](https://img.shields.io/npm/v/vtrack.svg)](https://www.npmjs.org/package/vtrack)
[![Downloads](https://img.shields.io/npm/dm/vtrack.svg)](https://www.npmjs.com/package/vtrack)

A biblioteca Vtrack Node fornece acesso conveniente à API do Vtrack.

## Instalação

Instale o pacote com:

```sh
npm install vtrack
# or
yarn add vtrack
```
